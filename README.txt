Варианты клиентских запросов:

		get_state
		set_state on | off
		get_color
		set_color red | green | blue	
		get_rate
		set_rate  0...5


Замечания по логике работы сервера:

		При запросе на чтение/изменение цвета или частоты, когда светодиод находится
		в состоянии "off" будет возвращено значение "FAILED".
	
		При переводе светодиод  из состояния "on" в состояние "off": color=red, rate=0.
  		При переводе светодиод  из состояния "off" в состояние "on": color=unknown, rate=unknown.


Все используемые FIFO создаются в каталоге /tmp.
